# log4j2-client
#### 项目介绍
简单日志记录工具

#### 使用说明
##普通Java引入日志
```
<repositories>
  <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
  </repository>
</repositories>
  
<dependency>
    <groupId>com.github.doobo</groupId>
    <artifactId>log4j2-client</artifactId>
    <version>1.2.1</version>
</dependency>
```

#spring-mvc引入日志
```
<repositories>
  <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
  </repository>
</repositories>
  
<dependency>
  <groupId>com.github.doobo</groupId>
  <artifactId>log4j2-client</artifactId>
  <version>v1.0.mvc</version>
</dependency>
```
